Política de Privacidad

La presente Política de Privacidad describe cómo se tratan los datos en este sitio web de acuerdo con el Reglamento General de Protección de Datos (RGPD).

1. Datos que se recopilan

Este sitio web está diseñado como una herramienta interactiva estática. Los datos introducidos en los simuladores, asistentes de requisitos y calculadoras de plazos (como fechas de entrada al país) se procesan localmente en el navegador del usuario y no son almacenados, guardados ni enviados a ningún servidor externo.

2. Proveedores de Terceros (Google AdSense)

Este sitio utiliza Google AdSense para mostrar anuncios. Google, como proveedor asociado, utiliza cookies para publicar anuncios basados en las visitas previas de los usuarios a este u otros sitios web. Los usuarios pueden inhabilitar la publicidad personalizada visitando la Configuración de anuncios de Google.

3. Derechos del Usuario

Dado que no se almacenan datos personales ni registros de identidad directamente en el servidor de este sitio web, no se conservan ficheros de tratamiento de datos de carácter personal de los visitantes.
